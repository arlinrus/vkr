# Данные

  Эмпирической основой служит открытый набор данных Open E-Commerce 1.0 (MIT Media Lab, Harvard Dataverse, 2023; описан в Nature Scientific Data, 2024). Датасет содержит 1 850 717 транзакций 5 027 пользователей онлайн-платформы за период 2018–2022 гг. с привязкой к демографическим характеристикам. Корреляция данных выборки с официальной квартальной отчётностью платформы составляет r = 0,978 (p < 0,001). Датасет распространяется по лицензии CC BY 4.0.
